# Toy Robot Simulator

A simulation of a toy robot moving on a 5x5 grid.

## Setup
`npm i`

## Run
`npm start`

## Test
`npm test`

## Commands
PLACE X,Y,F  — place the robot (F: NORTH, SOUTH, EAST, WEST)
MOVE         — move one step forward
LEFT         — rotate 90° left
RIGHT        — rotate 90° right
REPORT       — print current position
